Usando links internos

Se você quiser pode acessar a minha segunda pagina clique aqui!

Para ir da oficial para uma segunda pagina usamos rel="next"

Esse documento é um oferecimento de Hostnet

Se voce quiser que não seja um "patrocionio" ou uma materia que voce nao tem certeza que é real utilize rel="nofollow"

Para ao mecanismo do google saber que é na propria aba voce usa o comando target="self"

Outro atributo bem útil da tag de âncora é o hreflang="linguagem do site ", que permite indicar qual é o idioma principal do site para onde o link está desviando o fluxo de navegação. Isso vai permitir avisar ao navegador e a softwares de tradução como lidar caso o visitante opte por traduzir automaticamente os conteúdos

Definições a usar no (target="")

_blank
Vai abrir o link em uma nova janela em branco
_self
Vai abrir o link na janela ou frame atual (padrão)
_top
Vai desfazer todos os frames e abrir o destino no navegador completo
_parent
similar ao uso do _top em uma referencia a janela mão nome-do-frame caso esteja usando frames, indicar o nome da janela a abir

Definições a usar no (rel="")

next
Indica que o link é para a proxima parte do documento atual
prev
Indica que o link é para a parte anterior do documento atual
author
Indica que é um link para o site do autor do artigo atual
external
Indica que é um link para outro site que não faz parte do site
